Main Projects

  • Fancier Block Particles

    Fancier Block Particles is a Minecraft mod made by me which replace Minecraft boring 2D particles with 3D particles with physics.

    It is a fork of Fancy Block Particles that picked up back where the former developer of in 2018, but today the internals changed so much that not one line of code has not changed.
